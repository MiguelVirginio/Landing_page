# 🚀 Portfólio Pessoal — João Silva

Site portfólio desenvolvido com HTML5, CSS3 e JavaScript puro.

## 📁 Estrutura

portfolio/
├── index.html
├── sobre.html
├── projetos.html
├── contato.html
├── css/
│   ├── style.css
│   └── responsive.css
├── js/
│   └── script.js
└── img/

## ✅ Funcionalidades

- Menu responsivo com hamburguer
- Botão voltar ao topo
- Scroll Reveal (animação de entrada)
- Barras de habilidade animadas
- Formulário com validação JavaScript
- Efeito digitação no hero
- SEO básico em todas as páginas

## 🖥️ Como usar

1. Baixe ou clone o repositório
2. Abra o arquivo index.html no navegador
3. Não precisa de servidor ou instalação

## 🛠️ Tecnologias

- HTML5
- CSS3 (Flexbox + Media Queries)
- JavaScript ES6 (Vanilla)
